# 🛍️ MarketPlacer - Маркетплейс на FastAPI

Полноценный интернет-магазин с корзиной, избранным, админ-панелью и пагинацией.

## ✨ Возможности

- 🏠 Главная страница с каталогом товаров (15 товаров)
- 🔍 Поиск по названию и описанию
- 📂 Фильтрация по категории и цене
- 📄 Пагинация (9 товаров на страницу)
- 🛒 Корзина (хранение в куках)
- ❤️ Избранное (хранение в куках)
- 💬 Комментарии к товарам
- 📊 Статистика (количество товаров, средняя цена и т.д.)
- 👑 Админ-панель (управление товарами, комментариями, пользователями)

## 🚀 Быстрый старт

```bash
# Клонирование репозитория
git clone https://github.com/ваш-ник/marketplacer.git
cd marketplacer

# Создание виртуального окружения
python -m venv venv

# Активация (Windows)
venv\Scripts\activate

# Активация (Mac/Linux)
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt

# Запуск сервера
python main.py
